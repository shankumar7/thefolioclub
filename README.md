# thefolioclub
The premier open-source gallery of high-quality developer portfolios, creative templates, and personal site scaffolds. Built by the community, for the community.

The Folio Club is a curated, open-source ecosystem designed to eliminate "blank page anxiety" for developers, designers, and students. We bridge the gap between high-performance code and stunning design by collecting, vetting, and showcasing the web's finest personal portfolios.
