# How to use this template

1. Copy this whole folder to `f1/<your-github-username>-<short-name>`.
2. Edit `portfolio.json` — every field matters, see the root CONTRIBUTING.md.
3. Replace `index.html` / `style.css` / `script.js` with your real portfolio
   (or delete them and follow the `prebuilt` or `external` path instead — see
   CONTRIBUTING.md for all three options).
4. Replace this README with a real `preview.png` (1200×750 recommended).
5. Delete this file before opening your pull request.
